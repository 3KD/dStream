### Promotional Text (≤ 170 chars)

### Keywords (≤ 100 chars)

`decentralized,streaming,live video,nostr,monero,p2p,creator,non-custodial,privacy`

### Description

`dStream is a decentralized live-streaming app built for independent creators and communities.

Unlike traditional platforms, dStream does not require a single central host for identity and discovery. Stream metadata and social coordination are relay-native, and media delivery supports low-latency and fallback playback paths.

What you can do:
- Go live with camera and microphone from mobile.
- Watch live streams and jump into chat in real time.
- Manage your profile, follows, inbox, and guild/community presence.
- Use non-custodial wallet flows for supported payment rails.
- Report abuse in-app and apply moderation controls on official discovery surfaces.

Safety and moderation:
- In-app report flows for stream, user, and message targets.
- User-level mute and block controls.
- Policy pages are available in-app: Terms, Privacy, and Community Guidelines.

Non-custodial by design:
- dStream does not hold private keys or custody user funds.
- Node operators are responsible for content they publish or relay on their own infrastructure.`

## Google Play (Android)

### App name

`dStream`

### Short description (≤ 80 chars)

`Decentralized live streaming with non-custodial identity and relay-native discovery.`

### Full description

`dStream is a decentralized live-streaming app for creators and communities that want direct control.

dStream uses relay-native identity/discovery and a replaceable media stack, so operators can run independently without mandatory central hosting.

Core capabilities:
- Start live broadcasts from mobile camera + microphone.
- Watch live streams with resilient playback modes.
- Participate in real-time chat and social discovery.
- Manage identity, profile, follows, inbox, and communities.
- Use non-custodial wallet-integrated payment flows.

Moderation and safety:
- In-app abuse reporting for streams, users, and chat messages.
- Mute/block tools for personal safety.
- Operator moderation controls for official discovery surfaces.
- Policy pages in-app: Terms, Privacy, Community Guidelines.

Important:
- dStream is non-custodial and does not hold user private keys.
- Content responsibility is node-local: operators are responsible for what they publish or relay.`
