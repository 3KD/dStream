# Visuals Kit (Landing Animations)

This repo keeps the “front page look” (logo + rotating cube + color-synced word) in a small, liftable set of files.

## Dev preview

- Run the web app, then open: `/dev/visuals` (dev-only route)

## What to copy (if you want the visuals elsewhere)

- Cube component + word list/colors: `apps/web/src/components/landing/RotatingCube.tsx`
- Hero wrapper (cube + “Streaming” word + CTAs): `apps/web/src/components/landing/LandingHero.tsx`
- CSS for the cube (`.machined-*`): `apps/web/app/globals.css`
- Logo asset: `apps/web/public/logo_trimmed.png`
